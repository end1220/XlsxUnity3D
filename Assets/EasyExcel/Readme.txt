
========= EasyExcel =========

EasyExcel is for importing excel files to u3d project by generating csharp and *.asset files.
It makes using the data from excel files very easily.


Using:
ExcelDataReader - A library for reading Microsoft Excel files (2.0-2007)https://github.com/ExcelDataReader/ExcelDataReader.
EPPlus - A .NET library that reads and writes Excel files using the Office Open XML format (xlsx). https://github.com/JanKallman/EPPlus.
